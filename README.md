# all-palindromes
C++ simple Console application that reads data from a spacific file and finds all palindromes in a that file.

##Features:
* The App reads the data from the file "My words" that contains some words separated by space, comma or 
full stop. 
* prints all substrings that are palindromes in the file.
* Writes the output to a file.

##Requirments :
* C++ IDE
* C++ compiler
