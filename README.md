# all-palindromes
C++ small application that reads data from a spacific file and finds all palindromes in a that file.

Features:
1- The App reads the data from the file "My words" that contains some words separated by space, comma or 
full stop. 
2- prints all substrings that are palindromes in the file.
3- Writes the output to a file.

Requirments :
C++ IDE
C++ compiler
